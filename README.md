# Prism & Pigment - Vivid Color Formulation Tool

Prism & Pigment is a client-facing consultation tool used by stylists to design vivid hair color formulas during appointments.

The application allows stylists to mix pigments using sliders and visualize the resulting color on different hair levels.

---

## Features
- Virtual pigment mixing palette.
- Ratio-based color blending
- Hair level simulation (Level 10, 8, 6)
- Dynamix SVG hair visualization
- Printable formulation card
- Dark mode interface
- Tablet optimized responsive design

---

## Tech Stack
HTML5
CSS3 (Flexbox + Grid)
Vanilla JavaScript
SVG Garphics

---

2. cd prism-pigment
